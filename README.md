# Scheduler Application (C++)

## Overview
A modular console-based task scheduler built using C++. This project demonstrates object-oriented programming, clean code structure, and basic task management functionality.

## Features
- Add tasks
- View tasks
- Update tasks

## Project Structure
- src/ → implementation files
- include/ → header files
  
## Architecture
- Scheduler: manages task operations
- Task: represents a single task
- Date/Time: utility classes
  
## Technologies
- C++
- OOP
  
## How to Run
### Using g++
g++ src/*.cpp app/main.cpp -I include -o scheduler

### Using Visual Studio
1. Open .sln file
2. Build solution
3. Run project
