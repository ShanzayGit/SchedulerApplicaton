# Scheduler Application (C++)

## Overview
A modular console-based task scheduler built using C++. This project demonstrates object-oriented programming, clean code structure, and basic task management functionality.

## Features
- Add tasks
- View tasks
- Update tasks

## Project Structure
- source/ → implementation files
- include/ → header files

## Technologies
- C++
- OOP

## How to Run
g++ source/*.cpp -I include -o scheduler
./scheduler   # or scheduler.exe on Windows
